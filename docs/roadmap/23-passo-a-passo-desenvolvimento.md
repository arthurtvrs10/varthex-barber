# 23 — Passo a Passo de Desenvolvimento

## Estratégia

O projeto deve ser feito em etapas.

A ordem recomendada é:

1. documentação e ambiente;
2. backend base;
3. frontend base;
4. banco e Redis;
5. autenticação;
6. usuários e barbearias;
7. cadastros principais;
8. agenda;
9. bloqueios;
10. comissões;
11. dashboard;
12. estoque;
13. fila de espera;
14. recorrência;
15. WhatsApp CRM;
16. monetização SaaS.

## Regra de aprendizado

Para cada semana:

1. leia o guia sem código;
2. tente executar sozinho;
3. registre dúvidas;
4. compare com o guia de correção;
5. ajuste seu repositório;
6. faça commit no GitHub.

## Stack do desenvolvimento

- Java + Spring Boot no backend.
- Next.js no frontend.
- PostgreSQL no banco.
- Redis para cache, filas e jobs.
- Docker Compose para ambiente local.
