# Aprendizado semanal — Varthex Barber

Esta pasta contém arquivos separados para cada semana do projeto **Varthex Barber**.

Stack planejada:

- Backend: Java 21 + Spring Boot
- Frontend: Next.js + TypeScript
- Banco: PostgreSQL
- Cache/filas: Redis
- Ambiente local: Docker Compose

## Como usar

Abra apenas o arquivo da semana atual. Cada arquivo segue a ordem:

1. Guia prático sem código.
2. Tentativa de implementação.
3. Guia com código para correção no final.
