# 04 — Requisitos Funcionais

## RF01 — Login

O sistema deve permitir login com e-mail e senha.

## RF02 — Cadastro de barbearia

O SuperAdmin deve conseguir cadastrar uma barbearia.

## RF03 — Cadastro de barbeiros

O Admin deve conseguir cadastrar barbeiros vinculados à sua barbearia.

## RF04 — Cadastro de clientes

O Admin deve conseguir cadastrar clientes.

## RF05 — Cadastro de serviços

O Admin deve conseguir cadastrar serviços com preço, duração e status.

## RF06 — Agenda

O sistema deve exibir a agenda por dia, barbeiro e status.

## RF07 — Agendamento

O sistema deve permitir criar, cancelar, reagendar e concluir agendamentos.

## RF08 — Bloqueio de horário

O sistema deve permitir bloquear e desbloquear horários com motivo.

## RF09 — Fila de espera

O sistema deve permitir que clientes entrem em uma fila de espera para datas lotadas.

## RF10 — Comissão

O sistema deve calcular comissão por barbeiro conforme regra definida.

## RF11 — Estoque

O sistema deve controlar entrada, saída e venda de produtos.

## RF12 — Dashboard

O sistema deve exibir métricas diárias, semanais e mensais.

## RF13 — WhatsApp CRM

O sistema deve preparar mensagens automáticas para clientes, barbeiros e dono.

## RF14 — Planos SaaS

O sistema deve permitir definir planos e limites para barbearias.
