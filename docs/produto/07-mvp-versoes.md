# 07 — MVP e Versões

## MVP

A primeira versão deve conter:

- autenticação;
- cadastro de barbearia;
- cadastro de admin;
- cadastro de barbeiros;
- cadastro de clientes;
- cadastro de serviços;
- agenda;
- agendamento;
- bloqueio e desbloqueio de horários;
- comissão simples;
- dashboard básico;
- documentação no GitHub.

## Versão 2

- fila de espera;
- agendamento recorrente;
- estoque;
- venda de produtos;
- fechamento diário, semanal e mensal;
- comissão avançada.

## Versão 3

- CRM WhatsApp;
- menu WhatsApp;
- avaliação Google;
- lembrete de corte;
- fidelidade;
- planos para clientes.

## Versão 4

- SuperAdmin avançado;
- monetização SaaS;
- controle de dispositivos;
- módulos pagos;
- pagamentos online;
- deploy de produção.
