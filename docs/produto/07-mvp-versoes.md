# 07 — MVP e Versões

## Estratégia

O sistema deve ser criado por versões.

A primeira versão não deve tentar fazer tudo. O objetivo do MVP é validar o núcleo do produto: agenda, usuários, serviços, horários, agendamentos, comissão e dashboard básico.

---

# MVP — Versão 1

## Objetivo

Criar um sistema funcional para uma barbearia operar o básico.

## Recursos incluídos

- Login.
- Cadastro de barbearia.
- Cadastro de Admin.
- Cadastro de barbeiros.
- Cadastro de clientes.
- Cadastro de serviços.
- Cadastro de horários da barbearia.
- Horário individual por barbeiro.
- Agenda diária e semanal.
- Criar agendamento.
- Cancelar agendamento.
- Reagendar agendamento.
- Bloquear horário.
- Desbloquear horário.
- Marcar atendimento como concluído.
- Marcar cliente como não compareceu.
- Comissão simples.
- Dashboard básico.
- Fechamento diário.
- Notificações internas simples.

## Fora do MVP

- WhatsApp automático.
- Google Agenda.
- Fidelidade.
- Planos para clientes.
- Estoque avançado.
- Fila de espera avançada.
- Sistema completo de assinatura SaaS.
